# AI Resources for Documentation

This repository acts as a pool for our ongoing AI experiments in documentation.
It holds various re-usable prompts, LLM-generated scripts and other tools to
facilitate our work on documentation.

## Learning resources

### Generative AI, agents

- https://www.kaggle.com/learn-guide/5-day-genai - 5-Day Gen AI Intensive Course with Google and Kaggle (dated but relevant; explains how agents work on the inside)
  
- https://www.kaggle.com/learn-guide/5-day-agents - 5-Day AI Agents Intensive Course with Google (less dated, more specific to agentic applications)

### GitHub Copilot

- https://learn.github.com/skills, _Take flight with GitHub Copilot_ - an end-to-end collection of beginner-level practical exercises that leverage GitHub Copilot. Most approaches exercised there, however, are universally applicable.

- _GitHub Copilot Fundamentals_ ([part 1](https://learn.microsoft.com/en-gb/training/paths/copilot/), [part 2](https://learn.microsoft.com/en-gb/training/paths/gh-copilot-2/)) - Microsoft Learn training modules on Copilot; a more theoretical take on the basics.
  Most notably, this includes the [Documentation generation](https://learn.microsoft.com/en-gb/training/modules/generate-documentation-using-github-copilot-tools/) module.
